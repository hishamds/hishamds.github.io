## 🚀 How to Run Locally

1. Clone this repo:
   ```bash
   git clone https://github.com/hishamds/hishamds.github.io
   cd hishamds.github.io
   ```

2. Open `index.html` in any web browser.

> This is a static site — no server or build tools needed.
